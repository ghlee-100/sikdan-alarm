# 어린이집 식단 알림 서비스

매일 아침 어린이집 식단 정보를 카카오톡으로 자동 발송하는 알림 서비스입니다.

## 서비스 소개

어린이집 보호자가 매일 아침 자녀의 식단(오전간식, 점심, 오후간식)을 카카오톡 메시지로 받아볼 수 있습니다.

## 주요 기능

-  매일 오전 평일(월~금) 자동 발송
-  오전간식 / 점심 / 오후간식 정보 포함
-  카카오톡 메시지로 수신
-  월별 식단표를 JSON 파일로 관리

## 동작 방식

1. 매월 초 어린이집에서 제공하는 식단표를 `meal_data.json` 파일로 변환하여 업로드
2. GitHub Actions가 매일 오전 평일 자동 실행
3. 오늘 날짜에 해당하는 식단 정보를 카카오톡으로 발송

## 기술 스택

- Python 3
- GitHub Actions (자동 스케줄 실행)
- Kakao API (메시지 발송)

## 발송 메시지 예시

```
🍱 2026-04-22 어린이집 식단

🌅 오전간식:
떠먹는요거트

🍚 점심:
닭살카레라이스
쇠고기뭇국
호박나물
백김치

🌆 오후간식:
미니우동
```

## 대상

- 어린이집 재원 아동의 보호자
- 카카오톡 채널을 통해 구독 후 수신 가능
